# Cycles & Milestones

## Cycles

Cycles are time-boxed work periods (similar to sprints). The **Cycles** tab shows:

- **Active** cycles — Currently in progress
- **Upcoming** cycles — Planned for the future
- **Completed** cycles — Past cycles

Each cycle card displays:

- Cycle name and date range
- Issue count (completed / total)
- Progress bar

Click a cycle to see the issues assigned to it.

## Milestones

The **Milestones** tab shows key deadlines across projects. Each milestone displays:

- Milestone name
- Parent project (with color indicator)
- Target date (overdue dates are highlighted in red)

## Initiatives

The **Initiatives** tab shows high-level strategic goals that span multiple projects. Each initiative displays:

- Initiative name and status
- Number of linked projects
- Target date (if set)
